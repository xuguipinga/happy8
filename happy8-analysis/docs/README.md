# 快乐8彩票分析系统

这是一个基于 Flask 的快乐8彩票数据分析和可视化系统。通过对历史开奖数据的统计分析，提供号码遗漏、冷热分布、模式识别和智能推荐等功能，帮助用户更好地理解彩票数据趋势。

## 主要功能

-   **数据分析大屏**：直观展示历史数据和分析摘要。
-   **遗漏分析**：统计每个号码的当前遗漏期数。
-   **热号分析**：基于最近20期的号码出现频次统计。
-   **模式识别**：自动识别连号、同尾号、AC值等高级统计特征。
-   **智能推荐**：结合遗漏和热度，生成多种选号策略（选四、选五、选十等）。
-   **数据录入**：支持手动录入开奖数据，实时更新分析。
-   **智能更新**：每天开奖后（约 21:45）自动尝试更新最新数据。

## 快速安装

1.  **克隆或下载项目**
2.  **创建虚拟环境并安装依赖**
    ```bash
    python -m venv venv
    source venv/bin/activate  # Linux/Mac
    venv\Scripts\activate     # Windows
    pip install -r requirements.txt
    ```
3.  **运行程序**
    ```bash
    python run.py
    ```
4.  **访问系统**
    打开浏览器访问 `http://127.0.0.1:5000`

## 技术栈

-   **后端**：Python / Flask
-   **前端**：HTML / CSS / JavaScript (原生 Vanilla JS)
-   **数据存储**：CSV (开奖历史) / JSON (分析记录)
-   **部署**：Docker / Nginx / Gunicorn (推荐)
